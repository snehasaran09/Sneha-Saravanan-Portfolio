# Sneha Saravanan — Portfolio (v2)

## Structure
```
index.html       ← Landing page (3D particles, path selector)
product.html     ← Product & Strategy resume
ml.html          ← ML / AI / Data resume  (includes RedisLite project)
css/shared.css   ← Shared styles
js/shared.js     ← Shared JS (cursor, canvas, scroll)
assets/          ← PDF resumes for download
netlify.toml     ← Netlify config
```

## Deploy to Netlify
1. Go to app.netlify.com/drop
2. Unzip and drag the `sneha-portfolio-v2` folder onto the page — live in seconds

## Add Your Photo
Replace `<div class="avatar-placeholder">SS</div>` in `index.html` with:
```html
<img src="assets/photo.jpg" alt="Sneha Saravanan" />
```
